# Aplicación web

Aplicación web que analiza una imagen para saber si tiene un auto y determina si este presenta daños visibles o se encuentra en buen estado.

Utiliza React en el frontend y la API de OpenAI para el reconocimiento de imágenes.

## Características

- Carga una imagen desde el navegador
- Conversión automática de imagen a Base64
- Análisis de la imagen con OpenAI
- Detecta:
  - Si la imagen contiene un auto
  - Si el auto está en buen estado o dañado
  - Si el auto esta dañado, describe esos daños visibles

## Tecnologías utilizadas

- React
- JavaScript
- OpenAI API
- Vercel (deploy)
- HTML5
- CSS3
